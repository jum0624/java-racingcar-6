# 👩🏻‍💻 기능 설계

- 협력 : 초간단 자동차 경주 게임

## flow 살펴보기

1. 자동차 이름들을 입력한다
2. 시도할 횟수를 입력한다.
3. 시도한 횟수만큼 자동차들을 동작시킨다.
4. 동작 시키기 전 각 자동차의 랜덤수를 돌려 0-9사이에서 4이상이 나오면 움직인다.
5. 4미만일 경우 정지한다.
6. 한 회차의 자동차들의 움직임을 출력한다.
7. 시도한 횟수만큼 움직은 자동차들의 결과를 출력한다.

## 역할과 책임 구분

- 입력
    - 하는 것
        - 자동차 이름 입력받기
        - 시도할 횟수 입력받기

    - 아는 것

- 자동차(들)
    - 하는 것
        - 앞으로 가기("-" 출력)
        - 정지하기

    - 아는 것
        - 자동차의 이름


- 자동차 동작
    - 하는 것
        - 랜덤 수를 돌려 0-9사이의 값 가져오기
        - 4이상일 경우, 자동차의 앞으로 가기 요청
        - 4미만일 경우, 자동차 정치하기 요청

    - 아는 것
        - 동작시킬 자동차 객체

- 출력
    - 하는 것
        - 한 회차마다의 자동차 움직임 출력
    
    - 아는 것
